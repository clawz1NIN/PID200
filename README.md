**PID200** — це потужна, зручна і сучасна програма для системної підготовки до **Національного мультипредметного тесту (НМТ)**.  

Програма створена спеціально для українських випускників, які прагнуть максимальних результатів. Тут ти можеш тренуватися в умовах, максимально наближених до реального іспиту, аналізувати помилки та бачити свій прогрес у реальному часі.

## ✨ Основні можливості

- **Повна симуляція НМТ** — проходь 4 предмети за один раз з таймером і психологічною адаптацією
- **Тисячі актуальних завдань** — усі типи тестів 2025–2026 років
- **Детальна аналітика** — графіки, статистика по темах і предметах
- **Тематичні та змішані тести** — тренуйся саме там, де потрібно
- **Сучасний дизайн** — темна та світла тема, приємний інтерфейс
